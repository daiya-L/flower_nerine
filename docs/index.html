concept.

何気ない「日常」にあなただけの「色」を。

あなたの日常を彩る花を提供いたします。
落ち着きたいとき、気分を楽しくしたいとき、
元気をもらいたいときなど、様々なシーンに
寄り添う花をお選び頂けます。

nerine.

「また会う日を楽しみに。」

店の名前の由来にもなっている「ネリネ」
(ダイヤモンドリリー)という花の花言葉は
「また会う日を楽しみに」。お客様がまた
このお店に来ていただけることを心より
楽しみにする、そんな意味が込められた
名前となっています。

service.

個性のある花達を「飾る」。

nerine.ではフラワーアレンジメント、
プレゼント用の花束作りなどのサービスを
行っています。お客様が選んだ様々な花を
美しく「飾って」みてはいかがでしょうか。

item.

それぞれの「言葉」。

ひとつひとつの花それぞれの色、形があり、
それぞれに花言葉などの意味があります。
花ごとの違いを楽しんでみてください。
